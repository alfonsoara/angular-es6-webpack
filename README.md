# angular1.5-ES6-Webpack

## Install it globally.

```
npm i -g webpack
```

## If you will work with webpack, install webpack-dev-server too

```
npm i -g webpack-dev-server
```

## Dev ( server )

```
npm run-script devserver
```

## Build

```
npm run-script build
```

## Open url

```
http://localhost:9100/
```
